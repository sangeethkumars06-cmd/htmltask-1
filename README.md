# Hello World HTML

## Assignment Description

This assignment is a simple HTML webpage that displays the text **"Hello World!"** on the browser.

## Files Included

* `index.html` – Contains the HTML code.
* `README.md` – assignment documentation.

## How to Run

1. Open the assignment folder.
2. Double-click `index.html` or open it in any web browser.
3. The browser will display:

   ```
   Hello World!
   ```
